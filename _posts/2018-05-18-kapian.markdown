## 正文
好，现在来说第四个任务。这个任务主要是运用CSS transform 3D 变形来实现一个卡片翻转效果。详情请参见[设计师学院任务四]  (http://ife.baidu.com/course/detail/id/31)
**下面放我今天写的代码**
```
<!DOCTYPE html>
<html>
<head>
    <title>卡片翻转</title>
    <style type="text/css">
        .wrap{
            width: 250px;
            height: 350px;
            position: absolute;
            margin: auto;
            left: 50%;
            top: 50%;
            margin-left: -125px;
            margin-top: -175px;
            perspective: 800px;
        }
        .box{
            width: 250px;
            height: 350px;
            position: absolute;;
            left: 0;
            border: 0px solid #fff;
            transform-style: preserve-3d;
            transform-origin: right center; 
        }
        .box img{
            display: block;
            width: 100%;
            height: 100%;
            position: absolute;
            top: 0;
            left: 0;
            backface-visibility: hidden;
        }
        .box img:nth-of-type(1){
            transition: transform .8s;
        }
        .box img:nth-of-type(2){
            transform: rotateY(-180deg);
            transition: transform .8s;
        }
        .box:hover > img:nth-of-type(1){
            transform: rotateY(180deg);
        }
        .box:hover > img:nth-of-type(2){
            transform: rotateY(0deg);
        }
    </style>
</head>
<body>
    <div class="wrap">
        <div class="box">
            <img src="img\正面.jpg" />
            <img src="img\反面.jpg" />
        </div>
    </div>
</body>
</html>
```
现在来说说具体实现，首先，这个demo的html结构如下
```
舞台
    容器
        图片
        图片
```
* 对于舞台，我们给它加个视距，800px
`perspective: 800px;`
* 对于容器（即那个施加动画的元素），加个3D视图说明
`transform-style: preserve-3d;`
* 对于图片们，加上图片元素旋转到背面不显示的效果
`backface-visibility: hidden;`
 * **正面图**，鼠标放上去之前我们保持其不变，鼠标放上去后顺时针旋转180°
 ```
  .box:hover > img:nth-of-type(1){
            transform: rotateY(180deg);
        }
  ```
  添加过渡属性`transition: transform .8s;`
  
  * **反面图**，鼠标放上去之前就逆时针旋转180°，鼠标放上去之后再旋转到0°
  ```
  .box img:nth-of-type(2){
            transform: rotateY(-180deg);
        }
 .box:hover > img:nth-of-type(2){
            transform: rotateY(0deg);
        }
  ```
  添加过渡属性`transition: transform .8s;`
